- combineReducers
- useDispatch, useSelector
- useParams for Routing

## Backend via local DB (_unfortunately_)

GET requests on `localhost:3000` via `json-server`

```sh
# important order
npx json-server src/db.json
npm start
```

POST requests on `jsonplaceholder` service
